Belajar Struct (C Programming Language)
Apasih itu Struct ?
Structure atau struct adalah kumpulan dari beberapa variabel dengan
beragam tipe data yang dibungkus dalam satu varabel.
Cara penulisan syntax struct sebagai berikut:
struct name_structure {
     dataType1 name_field1;
     dataType2 name_field2;
     â€¦

};
Berikut contoh dari penggunaan Struct:
struct pelanggan {
     char Name [50];
     char Id[50];
     char Address[50];
     char Date[50];
     char Gender[50];
     char Number[50];
    
};
Tipe data struct pelanggan telah dibuat. Ketika tipe struct dideklarasikan,
tidak ada penyimpanan atau memori yang dialokasikan. Pengalokasikan memori baru
dibuat ketika kita membuat variable atu object nya, untuk mengakses struct tersebut kita menggunakan dot syntax (.), Berikut cara membuat variable atau object dari struct:
int main (int argc, char *agrv[]){
     struct pelanggan r1;
     printf( "Masukan Nama Pelanggan: " );
     gets( r1.Name);
     printf( "Masukan ID Pelanggan: " );
     gets( r1.Id);
     printf( "Masukan Alamat Pelanggan: " );
     gets( r1.Address);
     printf( "Masukan Tanggal Lahir Pelanggan: " );
     gets( r1.Date);
     printf( "Masukan Jenis Kelamin Pelanggan: " );
     gets( r1.Gender);
     printf( "Masukan Nomor Telepon Pelanggan: " );
     gets( r1.Number);
    

     printf( "Nama Pelanggan: %s\n " , r1.Name );
     printf( "ID Pelanggan: %s\n " , r1.Id );
     printf( "Alamat Pelanggan: %s\n " , r1.Address );
     printf( "Tanggal Lahir Pelanggan: %s\n " , r1.Date );
     printf( "Jenis Kelamin Pelanggan: %s\n " , r1.Gender );
     printf( "Nomor Telepon Pelanggan: %s\n " , r1.Number );

     return 0 ;
};
